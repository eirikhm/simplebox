SimpleBox

Really easy, nice syntax

A regular modal appearing mid screen

var box = Simple.Box.Builder()
.setHeader("Awesome!")
.setContent("My first modal - all teh contents")
.addButton('Ok, I get it', 'btn-success')
.build();

Auto-resizing modal with a minimum size limit

var box = Simple.Box.Builder()
.setHeader("Resize!")
.setContent(content)
.addButton('close')
.setSetting('autoResize', true)
.setSetting('size', {min:400})
.build();
